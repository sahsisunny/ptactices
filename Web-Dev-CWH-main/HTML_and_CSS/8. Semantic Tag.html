8. Semantic Elements

What is Semantic Elements?

Semantic elements are those elements which are used to give meaning to the content. For example, <h1> is used to give heading to the content. <h1> is a semantic element. <div> is used to divide the content into different sections. <div> is not a semantic element.

Why Semantic Elements?

When we use semantic elements, it is easy for the search engine to understand the content. It is also easy for the screen reader to understand the content. It is also easy for the developer to understand the content.

What are the Semantic Elements?

There are many semantic elements. Some of them are:

  • <header>
  • <footer>
  • <nav>
  • <main>
  • <article>
  • <section>
  • <aside>
  • <details>
  • <summary>
  • <figure>
  • <figcaption>
  • <time>
  • <mark>
  • <progress>
  • <meter>
  • <dialog>
  • <data>
  • <output>
  • <canvas>
  • <audio>
  • <video>
  • <source>
  • <track>
  • <embed>
  • <object>
  • <param>
  • <picture>
  • <svg>
  • <math>
What is the difference between <div> and <section>?

<div> is a non-semantic element. <section> is a semantic element.
